# Appunti di Crittografia

Questa repository contiene i file **LaTeX** per gli *Appunti di Crittografia*, basati sul materiale del corso di Crittografia 

![GitHub release (latest by date)](https://img.shields.io/github/v/release/mirco0/AppuntiCrittografia?label=Ultima%20Versione)

La versione compilata del documento sarà disponibile per il download nella sezione **[Release](https://github.com/mirco0/AppuntiCrittografia/releases/latest)**

## Argomenti trattati

- **Sistemi crittografici**
  - Definizioni formali
  - Modelli di sicurezza
  - Tipi di attacchi
- **Segretezza e autenticità**
  - MAC
  - Schemi di firma
  - Segretezza perfetta
  - One-Time Pad
  - Teoremi di Shannon
- **Sicurezza semantica**
- **Crittosistemi classici**
  - Cifrario di Cesare (Shift)
  - Cifrario a sostituzione
  - Cifrario affine
  - Cifrario di Vigenère
  - Cifrario di Hill
  - Tecniche di crittoanalisi
- **Cifrari a blocchi**
  - Substitution-Permutation Network (SPN)
  - Feistel Network
  - Data Encryption Standard (DES)
- **Argomenti avanzati**
  - Campi finiti e gruppi
  - Funzioni hash
  - Scambio di chiavi
  - Curve ellittiche
  - RSA e fattorizzazione


#### Nota
Gli appunti potrebbero contenere errori di battitura o errori nei calcoli, è possibile aprire un [issue](https://github.com/mirco0/AppuntiCrittografia/issues) o inviare una pull request.
